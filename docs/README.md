# Documentation

This directory contains all documentation for the HFT Solana Trading System.

## Files

- `ARCHITECTURE.md` - System architecture and design decisions
- `DEPLOYMENT.md` - Deployment guides for different environments
- `TRADING_GUIDE.md` - Trading strategies and best practices

## Contributing

When adding new documentation, please follow these guidelines:

1. Use Markdown format (.md)
2. Include table of contents for longer documents
3. Use consistent formatting and structure
4. Keep documentation up-to-date with code changes
